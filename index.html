OCeLoT (odf toolkit command line tools)

About

  • An internal Domain-Specific Language for the Apache ODF Toolkit; specifically the ODF Toolkit’s Simple API. Written in jruby.

  • The central idea is to provide a simple idiomatic ruby syntax for performing common ODF tasks. Take a look at the examples to get an idea of what ocelot can do.

Getting Started

Installation

  1. Install java

  2. Install jruby

  3. git clone git://github.com/noah/ocelot.git

Hello, World

To test your installation, do the following:

% cd ocelot
% java -jar ocelot.jar examples/hello.rb

If everything worked, you should have a shiny new file, hello.odf, with a very special message inside.

Documentation

Read the friendly yardoc.

The examples should give a pretty good idea of what this library is capable of, although for the sake of simplicity they represent only a subset of the ODF Toolkit Simple API’s functionality. Extending the code to do more exotic things should be easy.

Development

Like most jruby projects, ocelot uses a combination of ruby code and java jars. To build the executable ocelot.jar, do the following from the top level directory:

% gem install warble
% rm -f ocelot.jar; warble

Please report any bugs on the issue tracker or email me; pull requests / patches welcome.
